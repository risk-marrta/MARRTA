## Application Flow - First Run

```
┌─────────────────────┐
│ User starts MARRTA  │
└──────────┬──────────┘
           │
           v
┌─────────────────────────────────────┐
│ Read marrta.ini settings            │
│ Check: disclaimerAccepted = ?       │
└──────────┬──────────────────────────┘
           │
           ├─── true ────────────────────────────────────┐
           │                                             │
           └─── false                                    │
                │                                        │
                v                                        │
       ┌─────────────────────┐                          │
       │ Show Disclaimer     │                          │
       │ Dialog with:        │                          │
       │ [Acepto] [No acepto]│                          │
       └──────────┬──────────┘                          │
                  │                                      │
        ┌─────────┴──────────┐                          │
        │                    │                          │
   [No acepto]          [Acepto]                        │
        │                    │                          │
        v                    v                          │
  ┌───────────┐    ┌──────────────────────┐            │
  │Exit app   │    │Save disclaimerAccepted│            │
  │(return 0) │    │= true to settings     │            │
  └───────────┘    └──────────┬────────────┘            │
                              │                         │
                              └─────────────────────────┤
                                                        │
                                                        v
                                              ┌──────────────────┐
                                              │Show MainWindow   │
                                              │Application starts│
                                              └──────────────────┘
```

## Menu Integration

```
Help Menu (menuAyuda)
├── Ayuda
├── ──────────────────
├── Comprobar Actualizaciones
├── Comprobar Nuevos Modelos  
├── ──────────────────
├── Enviar Comentario
├── Enviar Modelo Riesgo
├── ──────────────────
├── Acerca de...
├── Aviso legal           → Opens DlgDisclaimer (Full disclaimer)
└── Licencia de uso       → Opens DlgLicense (GPLv3 + link)
```

## Settings File (marrta.ini)

```ini
[General]
language=es
debugMode=false
createBackup=false
disclaimerAccepted=true    ← New setting added
lastPraxis=/path/to/last/file.pbd
recentFiles=/path/to/file1.pbd,/path/to/file2.pbd
databaseFolder=/path/to/databases
reportFolder=/path/to/reports
```
